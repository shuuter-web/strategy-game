# GitHub Pages 用（分割版）

## 置き方（最短）
1. このフォルダ一式を GitHub のリポジトリ直下にアップロード
2. リポジトリの **Settings → Pages**
   - Source: Deploy from a branch
   - Branch: main (or master)
   - Folder: /(root)
3. 表示されたURL（例: https://<user>.github.io/<repo>/ ）をスマホで開く

## 構成
- index.html
- css/style.css（元の <style> を移動）
- js/main.js（元の <script> を移動）
